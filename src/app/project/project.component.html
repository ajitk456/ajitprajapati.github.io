ONLINE CAR SHOPPING

Title: Online Car Shopping
Team size: 2
Front End: Java
Back End: Xampp localhost
College: N.K.T.T. College
Project description:
The aim of this project is to reduce the manual work. For staff members Online Car Shopping is a online web application. Through this website customers can search for a car from showrooms. Purchasable amount and others functional properties of the cars they able to see with the help of this website. Customers are able to deal with the dealer as well to purchase their desirable cars.
